# TARBOO Deobfuscate

مشروع لفك تشفير أكواد Node.js المشفرة باستخدام خوارزميات WebCrack محليًا فقط.

## كيفية التشغيل محليًا

1. ثبت الحزم:

```
npm install
```

2. شغل المشروع:

```
npm run dev
```

ثم افتح المتصفح على `http://localhost:3000`

## النشر على Vercel

- اربط المستودع بـ Vercel واضغط Deploy مباشرة.
